1. 컴파일하기전에 선행처리( #으로 시작문장 #define , #include 등 )를 담당한다. 

2. <iostream> 헤더 파일에 표준 입출력을 위한 클래스와 객체 등이 선언되어 있으며 따라서 cin 과 cout 또한 <iostream> 헤더파일에 포함되어 있다.

3. 다른 사람이 작성한 소스파일이나 목적 파일을 컴파일 또는 링크하는 경우가 있다.

4. 프로그램과 입출력 장치 사이에서 데이터 바이트의 흐름을 다루는 추상화된 통로이다.

5. stdout 이며 FILE*을 사용한다. <stdio.h> 헤더 파일에 정의되어 있으며, 파일 포인터를 통해 데이터를 스트림으로 연결한다.

6. cout 이며 <iostream> 헤더를 통해 제공된다.
