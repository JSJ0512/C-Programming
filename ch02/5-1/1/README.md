- 매개변수는 함수가 입력값을 받기 위해 선언해 두는 변수 , 인자는 함수를 호출할 때 실제로 전달하는 값

- 매개변수에 인자값 전달과 호출한 곳으로 반환

- 함수 호출시 실제 변수값이 아닌 복사된 값이 전달되어 별도의 메모리 공간에 저장되기 때문임

- 함수 외부의 변수 값을 변경해야 할 때 , 큰 구조체나 배열을 인자로 전달할 때

- 주소에 의한 호출이 더 좋다. 왜냐하면 메모리 효율성 및 성능 향상 및 일관된 데이터 상태 유지가 가능하며 같은 객체를 공유하여 사용할 수 있어 효율적이기 때문임

- 지역 변수의 메모리 해제와 호출된 함수에서 호출한 함수로의 제어권 복귀가 있음

- 함수 내에서 생성된 지역 변수가 함수 종료와 함께 메모리에서 사라지기 때문임
