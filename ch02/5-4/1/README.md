- 일반 생성자는 객체를 새로 만들때 초기화하는 역할함, 복사 생성자는 이미 생성된 동일한 클래스 타입의 다른 객체를 인자로 받아 새로운 객체를 복사하여 생성함

- 얇은 복사 생성자는 객체가 새로 생성될 때 다른 객체로 초기화하면 호출되며 멤버 값을 그대로 복사하며 포인터는 주소값만 복사하므로 같은 메모리를 공유험 , 대입연산은 이미 생성된 객체에 다른 객체 값을 대입할 때 호출되며 기존 자원을 정리한 뒤 데이터를 복사하여 대입하며 기본적으로 얕은 복사 방식으로 동작함

- 기본 생성자 , 복사 생성자 , 복사 대입 연산자 , 소멸자 가 있음
